# EtxeSafe APP 🏠🛡️

**EtxeSafe** es una solución innovadora de diseñada para agilizar la gestión de siniestros de hogar (cristales, vitrocerámicas y espejos) mediante el uso de formularios dinámicos y validación por imagen.

## 🚀 Características principales
* **Login Seguro:** Acceso personalizado para clientes.
* **Dashboard de Siniestros:** Visualización en tiempo real del estado de tus partes (Pendiente, En Revisión, Resuelto).
* **Formularios Dinámicos:** El sistema solo pregunta lo necesario según el objeto dañado (estructura Atributo-Valor).
* **Captura de Evidencias:** Interfaz de cámara integrada para reportar daños visualmente.

## 🛠️ Stack Tecnológico
* **Frontend:** React.js con TypeScript.
* **Tooling:** Vite (para un desarrollo ultra rápido).
* **Estilos:** Tailwind CSS / PostCSS.
* **Diseño:** Prototipado en Figma y exportado para fidelidad de UI.

## 📦 Instalación y Uso Local

Para ejecutar este proyecto en local, sigue estos pasos:

### 1. Requisitos Previos
Es necesario tener instalado **Node.js** (versión LTS recomendada).
* Si no lo tienes, descárgalo aquí: [nodejs.org](https://nodejs.org/)
* Para verificar si lo tienes, escribe `node -v` en tu terminal.

### 2. Clonar el repositorio
```bash
git clone https://github.com/samaantha6/EtxeSafeAPP.git
cd "EtxeSafe APP"
```

### 3. Instalar dependencias
Abre una terminal en la carpeta del proyecto y ejecuta:
```bash
npm install
```
Este paso descargará la carpeta node_modules necesaria para que el proyecto funcione.

### 4. Arrancar el servidor de desarrollo
Una vez terminada la instalación, inicia la aplicación con:
```bash
npm run dev
```

### 5. Ver la aplicación
```bash
La terminal te dará un enlace (normalmente http://localhost:5173).
```
